## Color

- Light/Dark modes are provided by `AppThemeProvider`.
- Name tokens aim to be **semantic** rather than raw hex values, e.g. `bg`, `bgPaper`, `fg`, `primary`, `error`.

> Keep UI contrast at least WCAG 2.1 AA. Use Storybook **A11y** panel to verify.

## Typography

- Base font-size is 16px, line-height 1.5.
- Heading scale follows MUI variants: `h1 ... h6`.
- Prefer semantic elements (`<h1>`, `<p>`, `<button>`) with MUI variants for visual style.

## Spacing & Radius

- Spacing unit: **8px**.
- Default radius: **8â€“12px** for interactive controls and surfaces.

## Elevation

- Use small shadows for cards and drawers.
- Prefer **borders + subtle shadow** for tables (see `agGrid.overrides.css`).

## Motion

- Keep motion subtle and purposeful (e.g., fade/slide on drawers).

## Theming ag-Grid

Quartz theme is enabled. We override a few CSS vars to match the DS:

```css
/* web/src/theme/agGrid.overrides.css */
:root {
  --ag-borders: solid 1px;
  --ag-border-color: rgba(0, 0, 0, 0.08);
  --ag-control-panel-background-color: transparent;
  --ag-row-border-style: solid;
  --ag-row-border-width: 1px;
  --ag-row-border-color: rgba(0, 0, 0, 0.06);
  --ag-grid-size: 6px;
  --ag-borders-critical: solid 2px;
  --ag-row-hover-color: rgba(0, 0, 0, 0.04);
}
.ag-theme-quartz,
.ag-theme-quartz-dark {
  --ag-card-radius: 12px;
  --ag-header-foreground-color: inherit;
}
```
